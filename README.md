# iMaker Restro  
### Modern Restaurant POS & Management System

iMaker Restro is a scalable, real-time Restaurant POS and Operations Management System designed to streamline daily workflows across restaurants, cafés, bars, and multi-floor dining establishments.

Built with performance, usability, and operational clarity in mind, the system digitizes everything — from staff access control to live order routing, billing, and analytics.

---

## Technology Stack

- **Frontend:** React JS  
- **Styling:** Tailwind CSS  
- **Icons:** Lucide React  
- **Build Tool:** Vite  

---

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/imaker-dev/restro-pos.git
cd restro-pos
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

# System Workflow Overview

iMaker Restro follows a structured operational flow aligned with real-world restaurant processes.

---

## 1. Staff Management

Create secure staff accounts with role-based access control.

**Supported Roles**
- Manager  
- Captain  
- Kitchen Staff  
- Bar Staff  
- Cashier  

**Features**
- Unique login credentials  
- Secure password protection  
- Staff identification code  
- Quick POS access via PIN  

---

## 2. Station Configuration

Define preparation stations for operational clarity.

**Capabilities**
- Kitchen, Bar, or Custom Stations  
- Printer assignment for automatic KOT/BOT generation  
- Staff-to-station mapping  
- Real-time order display  

---

## 3. Floor Management

Digitally map your restaurant layout.

**Features**
- Unlimited floor creation  
- Custom naming support  
- Editable layout configuration  
- Organized floor-based visibility  

---

## 4. Section Management

Organize seating zones within each floor.

**Examples**
- AC Section  
- Non-AC Section  
- Family Section  
- Smoking Area  

**Benefits**
- Multiple sections per floor  
- Smart order routing  
- Section-based visibility control  

---

## 5. Table Management

Configure tables with operational flexibility.

**Capabilities**
- Table name or number  
- Seating capacity  
- Table shape  
- Merge table functionality  
- Live availability status  

---

## 6. Tax Configuration

Flexible tax structure management.

**Supported**
- GST (5%, 12%, 18%)  
- VAT  
- Custom tax groups  

**Features**
- Product-level tax assignment  
- Automatic CGST and SGST calculation  

---

## 7. Menu Categorization

Organize products by service type.

**Service Modes**
- Bar  
- Restaurant  
- Both  

**Benefits**
- Role-based menu visibility  
- Clean product structure  
- Flexible category assignment  

---

## 8. Add-On Management

Create customizable add-on groups for products.

**Examples**
- Toppings  
- Extra sides  
- Beverage add-ons  

**Features**
- Unlimited add-on items  
- Individual pricing  
- Product-level linking  

---

## 9. Product Management

Centralized menu creation and control.

**Capabilities**
- Product details (name, description, price)  
- Veg / Non-Veg / Egg classification  
- Station assignment  
- Variant support  
- Add-on attachment  

---

## 10. Order Management

Real-time digital order processing.

**Process Flow**
- Captain places order via mobile interface  
- Automatic routing to respective station  
- Instant KOT/BOT printing  
- Kitchen marks items ready  
- Live status notification to captain  

---

## 11. Billing & Checkout

Streamlined payment and order completion.

**Features**
- Bill generation  
- Multiple payment modes (Cash, Card, UPI, etc.)  
- Order closure  
- Automatic table reset  

---

## 12. Takeaway Orders

Dedicated counter-based ordering flow.

**Highlights**
- No table required  
- Direct cashier management  
- Full kitchen ticket generation  
- Fast checkout process  

---

## 13. Reports & Analytics

Comprehensive business insights and performance tracking.

**Reports Include**
- Sales summaries  
- Staff performance  
- Tax breakdown  
- Shift reports  
- Date-based filtering  

---

## 14. System Settings

Administrative configuration panel.

**Controls**
- Outlet configuration  
- System preferences  
- Role-protected access  
- Instant application of updates
